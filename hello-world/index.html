博客的第一次整顿

发表于2022-04-05|更新于2022-07-09|生活
|阅读量:

本篇记录的主要是从语雀上讲所有笔记迁移到jkelly生成的博客上的整个心路历程，虽然说确实用学到的python完成了图片文件爬取下载和文件重命名以及文件的正则匹配等内容，但也深刻意识到了自己的编程水平仍有待提高，从准备开始做到最终完成花了整整一夜，效率属实有些堪忧。

一次又一次迈向成功的伟大尝试，结果却并没有成功，生活啊，真的很迷人呢
一次又一次的去试探，最后终于把这个网站的博客写作摸了个差不多，虽然还没学过前端开发，但是这么点东西也够自己日常使用了。 

import os
import re
from urllib.request import urlretrieve
from IPython.core.interactiveshell import InteractiveShell
InteractiveShell.ast_node_interactivity = 'all'
test_path='./测试文件夹'
copy_path='./拷贝文件夹'
for i in os.listdir(test_path):

    file_path=os.path.join(test_path,i)
    #对前缀进行适当处理
    qz='''---
title: '''+i.split('.')[0]+'''\ndate: 2022-03-26 10:34:00 +0800
categories: [机器学习]
tags: [python机器学习]
pin: false
author: 爱编程的小明

toc: true
comments: true
math: true
mermaid: true

---\n\n'''
    i=''+i
    flag=0
    file_copy_path=os.path.join(copy_path,i)
    if os.path.exists('./_image/'+i.split('.')[0].replace(' ','')):
        pass
    else:
        os.mkdir('./_image/'+i.split('.')[0].replace(' ',''))
    with open(file_path,'r',encoding='utf-8') as f , open(file_copy_path,'w',encoding='utf-8') as c:
        data=f.read()
        #正则匹配，将公式环境进行修正
        data=re.sub(r'\n\$',r'\n\n$$',data)
        data=re.sub(r'\}\$',r'}$$\n',data)
        #添加前缀
        data=qz+data
        img_all=re.findall(r'!\[(.*)\]\((.*)\)',data)
        print(len(img_all))
        if img_all:
            for img in img_all:
                if img[0].split('.')[1] in ['jpg','png','bmp']:
                    img_name=img[0].replace(' ','')
                    img_path='/_image/'+i.split('.')[0].replace(' ','')+f'/{flag}.'+img[0].split('.')[1]
                    flag+=1
                    img_url=img[1]
                    if not(os.path.exists('.'+img_path)):
                        a=urlretrieve(img_url, '.'+img_path)
                    data=re.sub(img_url,img_path,data)
        c.write(data)
