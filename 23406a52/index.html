Seabornеә“з®Җд»Ӣ

example gallery

Seaborn, a statistical graphics library created by Michael Waskom. Seaborn simplifies creating many common visualization types
Unlike when using matplotlib directly, it wasnвҖҷt necessary to specify attributes of the plot elements in terms of the color values or marker codes. Behind the scenes, seaborn handled the translation from values in the dataframe to arguments that matplotlib understands. This declarative approach lets you stay focused on the questions that you want to answer, rather than on the details of how to control matplotlib.

ж•ЈзӮ№еӣҫзҹ©йҳө

еңЁжҺўз©¶еҸҳйҮҸд№Ӣй—ҙе…ізі»зҡ„ж—¶еҖҷжҲ‘д»¬з»ҸеёёйңҖиҰҒжҹҘзңӢеҸҳйҮҸд№Ӣй—ҙзҡ„ж•ЈзӮ№еӣҫпјҢSeabornжҸҗдҫӣдәҶдёҖдёӘpairplotеҮҪж•°жқҘж–№дҫҝзҡ„иҝӣиЎҢиҝҷдёӘж“ҚдҪңпјҢиҜҘеҮҪж•°дјҡиҝ”еӣһжүҖжңүеҸҳйҮҸд№Ӣй—ҙж•ЈзӮ№еӣҫд»ҘеҸҠеҚ•дёӘеҸҳйҮҸзҡ„жҰӮзҺҮеҜҶеәҰдј°и®ЎжҲ–иҖ…зӣҙж–№еӣҫгҖӮ

д»ҘеҚ•дёҖеҸҳйҮҸеҸ–еҖјеҲҶжғ…еҶөзҡ„з»ҳеӣҫпјҲfaceted plotsпјү

жңүдәӣжғ…еҶөдёӢжҲ‘зҡ„ж•°жҚ®жңүйўқеӨ–зҡ„еҲҶз»„з»ҙеәҰпјҢиҝҷдёӘж—¶еҖҷе°ұйңҖиҰҒжҲ‘д»¬жңүдёҖдёӘеҮҪж•°иғҪй’ҲеҜ№иҜҘз»ҙеәҰзҡ„дёҚеҗҢеҸ–еҖјеҲҶеҲ«з»ҳеҲ¶дёҚеҗҢзҡ„еӣҫзүҮпјҢиҝҷдёӘж—¶еҖҷе°ұз”ЁеҲ°дәҶsnsзҡ„factorplotеҮҪж•°пјҡ

sns.factorplot(x=,y=,hue=,col=,row=,kind=,data=)

е…¶дёӯcolе’ҢrowиЎЁзӨәеӯҗеӣҫеҲҶзұ»зҡ„дҫқжҚ®пјҢhueеҸӮж•°иЎЁзӨәжҜҸдёҖе№…еӣҫеҶ…з»ҳеӣҫеҲҶејҖзҡ„ж ҮзӯҫпјҲеӣҫдҫӢеҪўжҲҗдҫқжҚ®пјүгҖӮ
kindиЎЁзӨәеӯҗеӣҫзұ»еһӢ

sns.factorplot(x='day', y='tip_pct', row='time',
               col='smoker',
