Python дёӯдёҺж–Үд»¶и·Ҝеҫ„жңүе…ізҡ„жЁЎеқ—жҳҜ os жЁЎеқ—

и·Ҝеҫ„иҺ·еҸ–

  • еҸ–еҫ—еҪ“еүҚе·ҘдҪңзӣ®еҪ•пјҡos.getcwd()
  • еҸ–еҫ—з»қеҜ№и·Ҝеҫ„пјҡos.path.abspath(str)#str иЎЁзӨәж–Үд»¶еҗҚ
  • еҸ–еҫ—зӣёеҜ№и·Ҝеҫ„пјҡos.path.relpath(path,start)#дј еӣһд»Һ start еҲ° path зҡ„зӣёеҜ№и·Ҝеҫ„пјҢзңҒз•ҘеҲҷдјҡдј еӣһеҪ“еүҚе·ҘдҪңзӣ®еҪ•еҲ° path зҡ„зӣёеҜ№и·Ҝеҫ„

и·Ҝеҫ„жЈҖжҹҘ

  • exist(path):жЈҖжҹҘ path жҳҜеҗҰеӯҳеңЁ
  • isabs(path):жЈҖжҹҘжҳҜеҗҰжҳҜз»қеҜ№и·Ҝеҫ„
  • isdir(path):жЈҖжҹҘжҳҜеҗҰжҳҜж–Үд»¶еӨ№
  • isfile(path):жЈҖжҹҘжҳҜеҗҰжҳҜж–Үд»¶

й’ҲеҜ№ж–Үд»¶зҡ„ж“ҚдҪң

  • mkdir(path):е»әз«Ӣ path ж–Үд»¶еӨ№
  • rmdir(path):еҲ йҷӨ path ж–Үд»¶еӨ№пјҢдҪҶеҝ…йЎ»жҳҜз©әзҡ„ж–Үд»¶еӨ№
  • remove(path):еҲ йҷӨ path ж–Үд»¶
  • chdir(path):е°ҶеҪ“еүҚе·ҘдҪңж–Үд»¶еӨ№ж”№иҮі pathгҖӮ

иҺ·еҸ–зү№е®ҡж–Үд»¶еӨ№зҡ„еҶ…е®№ os.listdir()

еҸҜд»ҘдҪҝз”Ё listdir иҺ·еҸ–ж–Үд»¶еӨ№зҡ„еҶ…е®№пјҢжӣҙеёёз”Ёзҡ„дёҖз§ҚиҺ·еҸ–ж–№ејҸжҳҜз”Ё globгҖӮ

дҪҝз”Ё glob еҲ—еҮәеҢ№й…Қж–Үд»¶

globпјҲпјүеҮҪж•°дјҡдҪҝз”Ё Unix shell зҡ„и§„еҲҷжқҘеҢ№й…Қж–Үд»¶жҲ–иҖ…зӣ®еҪ•пјҢиҖҢдёҚжҳҜжӣҙеӨҚжқӮзҡ„жӯЈеҲҷиЎЁиҫҫејҸгҖӮ
е…·дҪ“и§„еҲҷеҰӮдёӢжүҖзӨәпјҡ

  • *пјҡеҢ№й…Қд»»ж„Ҹеӯ—з¬Ұ
  • пјҹпјҡеҢ№й…ҚдёҖдёӘеӯ—з¬Ұ
  • [!abc]пјҡеҢ№й…Қ aпјҢbпјҢc д»ҘеӨ–зҡ„жүҖжңүеӯ—з¬Ұ

дҪҝз”ЁйҖҡй…Қз¬Ұ""пјҢеҚі".txt"иҺ·еҫ—жүҖжңү txt жү©еұ•еҗҚзҡ„ж–Үд»¶пјҢ"ch*.*"еҲ—еҮәжүҖжңү ch ејҖеӨҙзҡ„ж–Үд»¶

йҒҚеҺҶж–Үд»¶ж ‘ os.walk()

os.walk()дёҺ os.listdir()зӣёдјјпјҢдёҚиҝҮеүҚиҖ…еҸҜд»Ҙеҫ—еҲ°еӯҗж–Үд»¶еӨ№зҡ„е…·дҪ“жғ…еҶөпјҢos.walk()иҝ”еӣһеҖјжңүдёүдёӘ:

  • еҪ“еүҚж–Үд»¶еӨ№еҗҚз§°(dirName)
  • еӯҗж–Үд»¶еӨ№еҲ—иЎЁпјҲsub_dirNamesпјү
  • ж–Үд»¶еҲ—иЎЁ(fileNames)
import os
for dirName,sub_dirName,fileName in os.walk("D:\OneDrive - stu.xjtu.edu.cn\lm\ж–ҮжЎЈ\еҸҜеҫ…"):
    print("еҪ“еүҚж–Үд»¶еӨ№:",dirName)
    print("еӯҗж–Үд»¶еӨ№еҗҚз§°:",sub_dirName)
    print("ж–Үд»¶еҗҚз§°:",fileName)

ж–Үд»¶/ж–Үд»¶еӨ№зҡ„еӨҚеҲ¶зІҳиҙҙ

os жЁЎеқ—еҸҜд»Ҙз”ЁдәҺж–Үд»¶жҲ–иҖ…ж–Үд»¶еӨ№зҡ„ж–°е»әпјҢиҮідәҺеҜ№ж–Үд»¶еӨ№жҲ–иҖ…ж–Үд»¶зҡ„зӣёе…іж“ҚдҪңпјҢshutil жЁЎеқ—жҸҗдҫӣдәҶй—®д»·е’Ңж–Үд»¶еӨ№зҡ„еӨҚеҲ¶з§»еҠЁе’ҢеҲ йҷӨзӯүж“ҚдҪңгҖӮ

ж–Үд»¶зҡ„еӨҚеҲ¶

ж–Үд»¶зҡ„еӨҚеҲ¶дё»иҰҒеҲ©з”Ё copy еҮҪж•°пјҡ
