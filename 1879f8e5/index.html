з®Җд»Ӣ

HTML жҳҜз”ЁжқҘжҸҸиҝ°зҪ‘йЎөзҡ„дёҖз§ҚиҜӯиЁҖгҖӮ

HTML жҢҮзҡ„жҳҜи¶…ж–Үжң¬ж Үи®°иҜӯиЁҖ (Hyper Text Markup Language)
HTML дёҚжҳҜдёҖз§Қзј–зЁӢиҜӯиЁҖпјҢиҖҢжҳҜдёҖз§Қж Үи®°иҜӯиЁҖ (markup language)
ж Үи®°иҜӯиЁҖжҳҜдёҖеҘ—ж Үи®°ж Үзӯҫ (markup tag)
HTML дҪҝз”Ёж Үи®°ж ҮзӯҫжқҘжҸҸиҝ°зҪ‘йЎө

HTML ж Үи®°ж ҮзӯҫйҖҡеёёиў«з§°дёә HTML ж Үзӯҫ (HTML tag):
HTML ж ҮзӯҫжҳҜз”ұе°–жӢ¬еҸ·еҢ…еӣҙзҡ„е…ій”®иҜҚпјҢжҜ”еҰӮ <html>
HTML ж ҮзӯҫйҖҡеёёжҳҜжҲҗеҜ№еҮәзҺ°зҡ„пјҢжҜ”еҰӮ <b> е’Ң </b>
ж ҮзӯҫеҜ№дёӯзҡ„з¬¬дёҖдёӘж ҮзӯҫжҳҜејҖе§Ӣж ҮзӯҫпјҢз¬¬дәҢдёӘж ҮзӯҫжҳҜз»“жқҹж Үзӯҫ
ејҖе§Ӣе’Ңз»“жқҹж Үзӯҫд№ҹиў«з§°дёәејҖж”ҫж Үзӯҫе’Ңй—ӯеҗҲж Үзӯҫ

htmlж–ҮжЎЈеҢ…жӢ¬htmlж Үзӯҫе’ҢзәҜж–Үжң¬пјҢhtmlж–ҮжЎЈд№ҹиў«з§°дёәзҪ‘йЎөгҖӮWebжөҸи§ҲеҷЁзҡ„дҪңз”ЁжҳҜиҜ»еҸ–HTMLж–ҮжЎЈпјҢе№¶д»ҘзҪ‘йЎөзҡ„еҪўејҸжҳҫзӨәеҮәжқҘгҖӮ

еёёз”Ёзҡ„htmlж Үзӯҫ

дёҖдёӘhtmlж–ҮжЎЈеӨ§жҰӮдјҡеҢ…жӢ¬д»ҘдёӢеҶ…е®№пјҢеӨҚжқӮзҪ‘йЎөдёҖиҲ¬дјҡеҢ…жӢ¬жӣҙеӨҡдёҚеҗҢзҡ„ж Үзӯҫд»ҘеҸҠеҜ№ж ҮзӯҫиҝӣиЎҢеұһжҖ§зҡ„и°ғж•ҙжқҘеҫ—еҲ°жӣҙеҠ дё°еҜҢзҡ„йЎөйқўгҖӮ

<html>
 <body>
   <h1>My First Heading</h1>

 </body>
</html>

  • pж ҮзӯҫеҶ…зҡ„ж–Үжң¬жҳҫзӨәдёәж®өиҗҪ
  • h1ж ҮзӯҫеҶ…зҡ„ж–Үжң¬иў«жҳҫзӨәдёәдёҖзә§ж Үйўҳ
